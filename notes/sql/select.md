# SELECT Statement

## Definition

The SELECT statement is used to retrieve data from one or more tables.

---

## Syntax

```sql
SELECT column_name
FROM table_name;
```

---

## Example

```sql
SELECT employee_name, salary
FROM employees;
```

---

## Output

Displays employee name and salary.

---

## Interview Question

**Q:** What is the purpose of the SELECT statement?

**Answer:** It is used to retrieve data from database tables.

---

## Real Project Example

In Oracle APEX reports, the SELECT statement is used as the source query for Interactive Reports, Interactive Grids, Cards, and Classic Reports.
